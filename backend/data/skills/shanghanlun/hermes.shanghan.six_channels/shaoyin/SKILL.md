# 少陰病 Skill

## 提綱（原文直述）
> 少陰之為病，脈微細，但欲寐也。
（SHL_SONGBEN_0281）

## 總括（chapter_level_induction，模型歸納）
少陰病提綱為脈微細、但欲寐；分寒化（四逆湯、真武湯輩回陽）與熱化（黃連阿膠湯育陰清熱），另有咽痛諸方與三急下證，死證條文最多，最重判別預後。

欲解時：從子至寅上
