# 太陰病 Skill

## 提綱（原文直述）
> 太陰之為病，腹滿而吐，食不下，自利益甚，時腹自痛。若下之，必胸下結鞕。
（SHL_SONGBEN_0273）

## 總括（chapter_level_induction，模型歸納）
太陰病提綱為腹滿而吐、食不下、自利益甚、時腹自痛；屬裏虛寒，當溫之，宜服四逆輩；兼表者仍可先桂枝湯解外。

欲解時：從亥至丑上
