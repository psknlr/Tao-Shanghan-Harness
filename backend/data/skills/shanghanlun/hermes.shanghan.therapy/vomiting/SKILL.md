# 吐法 Skill

## 適應（吐法）
邪實胸中、痰實在上脘者，其高者因而越之，瓜蒂散主之；虛家、亡血家禁用。
- 指徵：惡寒、發熱、自汗出、反不惡寒、胸中痞硬、氣上衝、胸有寒、目赤、溫溫欲吐、乾嘔
- 代表方：瓜蒂散
- 條文：SHL_SONGBEN_0120、SHL_SONGBEN_0121、SHL_SONGBEN_0166、SHL_SONGBEN_0264、SHL_SONGBEN_0324、SHL_SONGBEN_0338

## 適應（吐法）
宋本《辨可吐第十九》專篇法度（邪實胸中、痰實在上脘者，其高者因而越之，瓜蒂散主之；虛家、亡血家禁用。…）
- 指徵：大法，春宜吐。、凡用吐湯，中病即止，不必盡劑也。、病胸上諸實，胸中鬱鬱而痛，不能食，欲使人按之，而反有涎唾，下利日十餘行，其脈反遲，寸口脈微滑，此可吐、宿食在上脘者，當吐之。、病人手足厥冷，脈乍結，以客氣在胸中。心下滿而煩，欲食不能食者，病在胸中，當吐之。

## 禁例（禁吐）
太陽病惡寒發熱在表者不可吐；少陰病膈上有寒飲者當溫之不可吐。
- 少陽中風，兩耳無所聞，目赤，胸中滿而煩者，不可吐下，吐下則悸而驚。
- 少陰病，飲食入口則吐，心中溫溫欲吐，復不能吐，始得之，手足寒，脈弦遲者，此胸中實

## 禁例（禁吐）
